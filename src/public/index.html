Default Keys :

/comments (3) /todos (193) For more : follow Instructions

    Instructions

  • Add an array of object in db.json file to start using it
  • 2 keys are already created take example from those
  • to start the server :
  • open terminal in its origin/main folder
  • run node src/index.js

  • BY default PORT is: 8080
  • To change the PORT after node src/index.js write your PORT
  • eg: node src/index.js 3000
  • Recommended : Use Default port

    Usage :

  • Page -: http://localhost:8080/key/?page=1
  • Limit -: http://localhost:8080/key/?page=1&limit=10
  • PATCH | PUT | DELETE -Request
  • url will be: `http://localhost:8080/key/${id}`
  • GET -Request
  • for all-url will be: `http://localhost:8080/key`
  • for single item-url will be: `http://localhost:8080/key/${id}`
  • POST -Request
  • url will be: `http://localhost:8080/key`
